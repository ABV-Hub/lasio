Welcome to lasio's API documentation!
=====================================

``lasio`` is a Python package to read and write Log ASCII Standard (LAS) files, used for borehole/well data (e.g. geophysical/geological/petrophysical logs).

This documentation covers lasio's API.

For more information about installation and usage please take a look at the `README on GitHub <https://github.com/kinverarity1/lasio/blob/master/README.rst>`__.
